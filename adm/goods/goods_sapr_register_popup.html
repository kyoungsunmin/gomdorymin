쇼핑몰 선택
절상하 선택
가격인상하 (정액) 원
가격인상하 (정률) %
구분자 입력

절상하선택(판매가의 단위 설정 항목) :
대부분 사이트에서 10원 단위 판매가는 등록 자체가 안되는 경우가 있습니다.

가격인상하(정액)(판매가 일괄 금액 변경 항목) :
판매가를 일률적으로 올려야 하는 경우 사용하시면 됩니다. 예] 2,500원 올림으로 선택하면, 저장할 상품이 일괄 판매가에서 2,500원 올려서 반영됩니다.

가격인상하(정률) :
판매가 대비 정률인상하를 적용하고자 할때 사용하시면 됩니다. 예] 5% 입력할 경우 10,000 -> 10,500원 , -5% 입력할 경우 10,000 -> 9,500으로 적용됩니다.

구분자는 작업을 진행하는 사용자가 임의로 편리하게 기록하고, 상품연동시 구분자 검색을 통해 해당 작업자가 관리/구분 할 수 있는 용도로 사용합니다.(예:영희작업,070601,..)

가격인상하(정액)와 가격인상하(정률)은 동시 사용이 불가합니다.
