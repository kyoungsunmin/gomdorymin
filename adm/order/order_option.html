옵션별 판매 현황

해당 선택한 기간 동안의 주문정보에 각 옵션별 판매수량을 파악하실 수 있습니다.

해당 선택한 날짜 기간은 애드위너에서 가져오게 되는 주문등록일자 기준으로 검색이 됩니다.

한 상품의 옵션이 2개 이상일 경우 연두색으로 표기되어 보이게 됩니다.

쇼핑몰별/매입처별로 옵션별 판매수량 관리가 가능하며 상세보기로 확인이 가능합니다.

각 상태별 설명
신규주문 : 각 쇼핑몰에서 주문이 수집된 직후 상태의 신규주문
발주확인 : 신규주문(배송대기중)을 주문서 다운로드한 후 변경한 상태
송장등록 : 송장이 등록된 상태
송장전송완료 : 송장번호가 기입된 상품을 해당 쇼핑몰로 전송하여 완료된 상태
교환 : 교환처리로 신규주문 접수된 상태
